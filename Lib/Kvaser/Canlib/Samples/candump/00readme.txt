CANLIB Sample Program : candump.exe
===================================

This is a 32-bit Windows console mode program that dumps the traffic
on the CAN bus to the screen, or to a file.

The program is just intended as an example of how to use CANLIB. It's not
performance optimized in any way. It may, or may not, be useful
for you.

The program is written in C and has been tested with Borland and
Microsoft compilers.

Type 'candump -?' for brief instructions on how to use the program.
