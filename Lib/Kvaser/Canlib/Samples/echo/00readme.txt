CANLIB Sample Program : Canecho.exe
================================

This is a 32-bit Windows console mode program that receives CAN messages
and retransmits them, after increasing the message's identifier by one.

The program is just intended as an example of how to use CANLIB. It's not
performance optimized in any way. It may, or may not, be useful
for you.

The program is written in C and has been tested with Borland and
Microsoft compilers.

Type echo -?' for brief instructions on how to use the program.
