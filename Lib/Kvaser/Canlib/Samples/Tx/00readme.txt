CANLIB Sample Program : tx.exe
===================================

This is a 32-bit Windows console mode program which can transmit messages
on the CAN bus. It might be used as a traffic generator.

The program is just intended as an example of how to use CANLIB. It's not
performance optimized in any way. It may, or may not, be useful
for you.

The program is written in C and has been tested with Borland and
Microsoft compilers.

Type 'tx -?' for brief instructions on how to use the program.
