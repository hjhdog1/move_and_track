Support

If you have any problems, please read this file and check the help file. You might also want to have a look at our web site, www.kvaser.com.

The CANLIB SDK contains everything you need to develop programs for all our CAN hardware but it does not install the actual driver routines on your system. Installation packages for the device drivers are available on the Kvaser CD and from our web site (http://www.kvaser.com/download)
